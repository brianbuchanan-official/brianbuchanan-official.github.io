Case Study

Monte Carlo Investment Simulation

Forecasts a distribution of future portfolio values using random draws from an assumed return distribution (mean μ, volatility σ). Outputs percentile bands, sample paths, and an optional goal-probability check.

Open HTML Notebook Download CSV Output View Repo Folder

Problem

Point forecasts hide uncertainty. For planning (retirement, college, capital projects), we need to see a realistic range of outcomes and the likelihood of hitting target values by a given horizon.

Approach

  • Generate 1,000 simulated annual return paths with mean μ and volatility σ.
  • Compute yearly portfolio values, then extract 5th/25th/50th/75th/95th percentiles.
  • Overlay a few sample paths to illustrate path-dependence.
  • (Optional) Add monthly contributions, fees, and inflation for more realism.

Key Outputs

Percentile Bands

Shows downside risk & upside potential (5–95% range).

Median Trajectory

Most likely growth path under current assumptions.

Goal Probability

Share of simulations that reach the target value by year N.

Tech

Python NumPy Pandas Matplotlib Jupyter

Tip: The HTML notebook shows parameters at the top so you can see how μ, σ, horizon, and contributions change the distribution.
